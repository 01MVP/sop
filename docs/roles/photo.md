# 🎥 摄影 SOP

**一句话目标**：捕捉专注、协作与情绪瞬间，输出可用素材。

## 现场执行清单（活动当天）

### 到场准备（活动开始前）
- [ ] 拍空镜与物料（场地、易拉宝、奖品、签到台）。
- [ ] 检查电池与存储，准备备用设备。

### 活动中
- [ ] 覆盖签到、全景、特写、路演、颁奖、合影。
- [ ] 有条件时拍 15-30 秒花絮短视频（讨论/展示/庆祝）。
- [ ] 每小时挑 2-3 张发给宣发同步现场感。

### 活动后（24 小时内）
- [ ] 精选 30-50 张，简单调色。
- [ ] 上传相册/网盘并发群，同时给宣发。

## 注意事项与应急

### 关键标准
- 以情绪与协作为主线，少拍空洞摆拍。
- 横竖构图兼顾，方便不同平台使用。

::: tip 大型活动可拆分
可拆为摄影、摄像与剪辑三岗。
:::

### 常见问题与应急处理
| 问题 | 解决方案 |
| --- | --- |
| **光线不足/逆光** | 调整角度或开启 HDR，必要时借灯 |
| **电量/存储不足** | 及时更换电池/存储卡 |
| **关键环节错过** | 与主持人对齐时间点，提前就位 |

### 摄影锦囊
1. 横构图适合公众号，竖构图适合短视频平台。
2. 尽量隐形拍摄，避免打断参与者。
