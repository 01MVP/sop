# 🎤 分享嘉宾 SOP

**一句话目标**：在限定时间内讲清核心价值，帮助参与者理解与行动。

> 如仅为嘉宾分享，不承担技术支持与巡场，技术问题由 [技术导师 Mentor](/roles/tech-support) 处理。

## 现场执行清单（活动当天）

### 活动前（T-3 ~ T-1）
- [ ] PPT 初稿发主理人审核（深度/时长/广告比例）。
- [ ] 根据反馈精简到 15 分钟以内。
- [ ] 将 PPT 与 PDF 备份发送给现场中控。

### 活动当天
- [ ] 提前 30 分钟到场测试投屏与翻页笔。
- [ ] 分享 3 个重点，图多字少。
- [ ] 预留 2-3 分钟互动与答疑。

### 活动后
- [ ] 如需公开，提供 PPT/资源链接给主理人/宣发。

## 注意事项与应急

### 关键标准
- 站在听众视角讲怎么做，少讲概念。
- 超时先删非核心，再直接总结。

::: tip 大型活动可拆分
可设置嘉宾联络/彩排对接专员，负责资料收集与时间提醒。
:::

### 常见问题与应急处理
| 问题 | 解决方案 |
| --- | --- |
| **分享超时** | 中控提醒后立即压缩内容 |
| **PPT 无法播放** | 切换 PDF 备份或脱稿分享 |
| **观众冷淡** | 增加互动问题或案例 |
| **演示失败** | 切换预录视频或图片流程 |
