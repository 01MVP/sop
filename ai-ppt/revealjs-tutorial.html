Reveal.js 是什么？

一个用 HTML/CSS/JS 做幻灯片的开源框架。

  • 网页即 PPT，浏览器即可演示
  • 可版本管理，适合协作与复用
  • 支持主题、插件、动画和演讲者模式

适合人群

需要快速迭代、多人协作、能写一点 HTML 的小白。

三步快速开始

  1. 准备一个 HTML 文件（本文件就是示例）
  2. 引入 Reveal.js 的 CSS 与 JS
  3. 按结构写 <section> 幻灯片并初始化

小白建议：用本地静态服务预览，避免浏览器安全限制。

最小模板

<div class="reveal">
  <div class="slides">
    <section>第一页</section>
    <section>第二页</section>
  </div>
</div>

<script src="https://cdn.jsdelivr.net/npm/reveal.js@5/dist/reveal.js"></script>
<script>
  Reveal.initialize({ hash: true });
</script>

提示：hash: true 会把当前页同步到 URL，便于分享。

横向与纵向

Reveal.js 支持「左右」与「上下」两种层级。

<section>
  <section>上</section>
  <section>下</section>
</section>

逐条出现（Fragments）

  • 先出现第一条
  • 再出现第二条
  • 最后出现第三条

给元素加 fragment 类即可。

背景色示例

使用 data-background-color 给某页上色。

<section data-background-color="#0f172a">...</section>

演讲者备注

台上展示内容与台下备注分离。

<section>
  <h2>台上显示</h2>
  <aside class="notes">这是演讲者备注</aside>
</section>
说明：按 S 打开演讲者视图。

用 Markdown 写幻灯片

需要启用 Markdown 插件。

<section data-markdown>
  <textarea data-template>
    ## 标题
    - 列表一
    - 列表二
  </textarea>
</section>

代码高亮

Reveal.js 自带高亮插件，只需引入即可。

function hello(name) {
  return `Hello ${name}`;
}

console.log(hello("Reveal"));

初始化脚本（含插件）

Reveal.initialize({
  hash: true,
  slideNumber: true,
  plugins: [ RevealMarkdown, RevealHighlight, RevealNotes ]
});

插件脚本需在 Reveal.js 后加载。

小白友好操作清单

  • 本地预览：python -m http.server 8000
  • 打开：http://localhost:8000/revealjs-tutorial.html
  • 导出 PDF：加参数 ?print-pdf 再打印
  • 常用快捷键：←/→ 翻页，↑/↓ 纵向，Esc 总览
